about me

My name is Mateusz Gadecki. I am a second-year student in the field of computer science, specialization: Front-End Developer. I have always felt that it was easy for me to interact with the computer and various technologies, but it was only over a year ago that I decided to try programming. I started with learning the basics of C++ and Python but then I started with Frontend and it stole my heart. Web development has become my passion. In every free moment I learn, write code or wonder what can be improved or added to the pages that I am currently creating. HTML, CSS, JavaScript are no secret to me. In recent projects I learned the basics of the React framework and used Redux. Now I'm learning React hooks. My dream is to take my first job as a junior front-end developer because I think that nothing develops better than working with more experienced programmers on real projects for the clients. In order to get acquainted with the projects written by me, I invite you to my github. Enjoy my website!
